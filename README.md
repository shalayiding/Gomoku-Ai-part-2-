# Gomoku-Game
1 hour coding to create basic Gomoku Game

## How to
    1.install python 2.x or 3.x(recommended)
    2.Install pygame
    $pip install pygame
    3.Clone the repository:
    $git clone https://github.com/shalayiding/Gomoku-Game
    4.run Gomuku.py
    $python Gomuku.py
